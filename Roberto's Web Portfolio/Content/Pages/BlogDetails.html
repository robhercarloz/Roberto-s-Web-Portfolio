Project
This was a fun project that was meant to be an introduction to the use of MVC(Model-View-Controller), Entity Framework for user authentication management, SQL databases and slugs. This put to test my technical skills and ability to work through new problems.
Project Difficulties
This project really tested my skills architecting a workable solution. In the beginning of development, the first thing I did was start up an S1 azure server. With no plan for action, I wasted a good amount of time worth of work before I was forced to start over.
My Solution
To solve this struggle, I decided to kick it old school with some good ol’ Object Oriented Design and Analysis. Following a planning structure, I was able to improve my work efficiency and reach my set sprints.
